## 🚀 الميزات الرئيسية (Features)
- 🛒 **سلة تسوق ومفضلة ذكية** باستخدام تقنيات (AJAX).
- 📱 **تصميم متجاوب 100% (Responsive)** يعمل على كافة الأجهزة (جوالات، تابلت، شاشات كبيرة).
- 📊 **لوحة تحكم (Admin Panel)** احترافية لعرض الإحصائيات وإدارة الكتب، التصنيفات، الطلبات، والمدونات.
- 🎮 **نظام ألعاب تفاعلي** لزيادة تفاعل المستخدمين ورفع نقاطهم.
- 🔒 **نظام صلاحيات وحماية متقدمة** مع تشفير لكلمات المرور.
- 📂 مبني باستخدام **PHP & PostgreSQL** بأسلوب برمجي نظيف ومهيكل (OOP).
